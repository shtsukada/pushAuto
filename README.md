# 目的
Gitの下記コマンド自動化ツール
- git init
- git secrets --install
- git secrets --register-aws
- git remote
- git add
- git commit
- git branch
- git push

## 前提条件

以下のツールがインストールされている必要があります：

- git
- git-secrets（https://github.com/awslabs/git-secrets）

macOSの場合は以下でインストール可能です：
```bash
brew install git-secrets
```
